## Getting Started

1. **Enter the frontend:**
   ```bash
   cd frontend
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Set up API key:**
   Copy `.env.example` to `.env` and add your **Gemini API Key**.
4. **Launch:**
   ```bash
   npm run dev
   ```
